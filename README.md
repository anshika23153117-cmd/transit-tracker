# ✈️ Transit Tracker

A full-stack real-time flight tracking application built with React, Node.js, and MongoDB.

## 🚀 Live Links
- **Frontend:** [https://transit-tracker-frontend.vercel.app](https://transit-tracker-frontend.vercel.app)
- **Backend API:** [https://transit-tracker-backend.onrender.com](https://transit-tracker-backend.onrender.com)

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Leaflet (Maps), Tailwind CSS
- **Backend:** Node.js, Express, CORS
- **Database:** MongoDB
- **Data Source:** AirLabs API

## ⚙️ How to Run Locally
1. Clone the repository.
2. Install dependencies in both folders using `npm install`.
3. Set up your `.env` files (refer to the environment variables section).
4. Run the backend with `npm start` and the frontend with `npm run dev`.
